Blackjack Game Project
Description:
This is a simple console-based Blackjack game implemented in Python. The game follows standard Blackjack rules where the player aims to beat the dealer without exceeding a total card value of 21.

Features:

Player vs. Dealer: Play against a computer-controlled dealer.
Card Deck: Utilizes a standard deck of 52 playing cards.
User Input: Accepts user input for actions like Hit or Stand.
Scoring: Calculates and displays the total score for both the player and the dealer.
Win/Loss Logic: Determines the winner based on Blackjack rules.
Restart Option: Allows the player to restart the game after completion.
Project Structure:

blackjack.py: The main Python script containing the game logic.
main.py: The entry point for blackjack.py
README.md: Documentation and instructions for running the game.

How to Play:

Clone the repository: git clone https://github.com/Gadget622/blackjack.git
Navigate to the project directory: cd blackjack
Run the game: python blackjack.py
